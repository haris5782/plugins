Instalasi
- ekstrak paket plugins.tar.gz sehingga akan menghasilkan direktori ./plugins
- salin direktori hasil ekstrak ke ./admin/modules/ sehingga hasil akhirnya
  adalah direktori ./admin/modules/plugins/
- import data plugins.sql ke database
